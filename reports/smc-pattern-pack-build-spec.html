Features Breakdown

Feature 1: Fair Value Gaps (FVG) Detection

What is FVG? An unfilled gap in price action (usually between two candles) that the market tends to come back and fill.

User Options:

  • Minimum gap size (in points or %)
  • Show all gaps vs high-probability gaps only
  • Minimum number of bars before marking as "unfilled"
  • Color scheme for bullish/bearish gaps

Feature 2: Break of Structure (BOS) Detection

What is BOS? When price breaks a previous swing high or low, it signals institutional movement (potential trend change).

User Options:

  • Lookback period for swing identification (default 10 bars)
  • Show swing highs/lows on chart
  • Alert on break
  • Color for uptrend vs downtrend breaks

Feature 3: Order Blocks (Optional/Advanced)

What is OB? A previous support/resistance zone where institutions likely placed orders (accumulation/distribution).
